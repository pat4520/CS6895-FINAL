Portfolio Management Dashboard: A Scalable Grid-powered Streaming Data Grid and Charting Dashboard

PM Dashboard re-iterates on our CS-6893 idea, using the technologies and tools we have learned in this course: to tick real prices, current sentiments and up-to-the-minute predictions into a live dashboard. It uses techniques in python, alchemy, spark and pyCuda as learnt in this course.

The website template for Streaming Data Grid is built with Bootstrap 3. Make sure you're up to date with latest Bootstrap documentation!

Streaming Data Grid

The website template for Streaming Data Grid is built with UI Grid. Make sure you're up to date with latest Bootstrap 3, Underscore.js and HTML5 WebSocket documentation!

Visualizations

The visualizations page is ready to go with Allocated Assets and Product Holdings charts, now all that is need is to add the user's own custom stylings! There are more free themes over at HighCharts, that can be coded up and integrated into the dashboard using the Angular HighCharts controller!

Architecture

This template features many common pages that you might see on a business website. Pages include: about, contact, portfolio variations, blog, pricing, FAQ, 404, services, and general multi-purpose pages.
